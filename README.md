# 消費税 理論連想

消費税法の理論対策用ミニアプリです。

キーワードを入力すると、関連して思い出すべき規定・注意点・答案上の確認事項をカード形式で確認できます。スマホでの利用を前提に、HTML/CSS/JavaScriptだけで動くようにしています。

## 使い方

1. `index.html` をブラウザで開く
2. キーワード欄に `棚卸資産 免税事業者` のように入力する
3. 関連カードを確認する
4. `練習` タブでランダム出題を使う
5. `追加` タブで自分の暗記カードを増やす

追加したカードはブラウザ内に保存されます。端末を変える場合は、`一覧` タブの `書き出し` と `読み込み` を使って移せます。

## GitHub Pagesで公開する場合

このリポジトリをGitHubにアップロードしたあと、GitHubのリポジトリ画面で次の設定をします。

1. `Settings`
2. `Pages`
3. `Build and deployment`
4. `Source` を `Deploy from a branch`
5. `Branch` を `main`、フォルダを `/root` に設定

公開URLは通常、次の形式になります。

`https://ユーザー名.github.io/リポジトリ名/`

## 注意

このリポジトリには、理論テキスト・過去問・模試PDFなどの教材ファイルは含めないでください。アプリには学習用の短い要約カードだけを入れています。
